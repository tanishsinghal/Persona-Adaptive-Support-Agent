# Persona-Adaptive Customer Support Agent

This project is a simple customer support chatbot that adapts its responses
based on the detected customer persona.

## Supported Personas
- Technical Expert
- Frustrated User
- Business Executive

## Key Features
- Rule-based persona detection
- Keyword-based knowledge base retrieval
- Persona-specific response tone
- Escalation to human agent with context

## Escalation Logic
- Explicit request for human support
- Repeated frustration
- Business-related queries

## Limitations
- Rule-based detection may not cover all cases
- KB retrieval is keyword-based
